# BM25 тестирование поиска

Скрипт `test_bm25.py` позволяет быстро протестировать BM25 (keyword search) на вашей базе знаний.

## Использование

### Режим 1: с аргументом в командной строке

```bash
cd /Users/mikhail/Projects/RAGv2
source .venv/bin/activate
python -m retriever.test_bm25 "ваш запрос здесь"
```

### Режим 2: интерактивный (просит ввести запрос)

```bash
python -m retriever.test_bm25
# Вводите запрос и нажимаете Enter
```

## Формат результатов

```
# Ваш запрос
# Найдено чанков: N

## 1
## Имя файла.md
## Score: 3.4454
## Первые 100 символов содержания...

## 2
## Другой файл.md
## Score: 3.2260
## ...
```

Каждый результат показывает:
- **Номер** в топ-10
- **Имя файла** где найден чанк
- **Score** — релевантность по BM25 (выше = релевантнее)
- **Первые 100 символов** содержания чанка

## Примеры запросов

```bash
# Поиск по задачам
python -m retriever.test_bm25 "какие задачи"

# Поиск по медицине
python -m retriever.test_bm25 "медицина здоровье"

# Многословный запрос
python -m retriever.test_bm25 "как организовать встречу команды"
```

## Что показывает скрипт

- **Топ-10 чанков** по BM25 релевантности
- **BM25 скор** для каждого результата
- **Статистику** (мин/макс/средний score)
- **Скорость выполнения** обычно < 1 сек

## Технические детали

- BM25 — это **keyword search**, работает по терминам в тексте
- Не использует семантику (семантический поиск в `python -m retriever.search`)
- Основан на `FastEmbedSparse` из LangChain
- Топ-10 по умолчанию, можно изменить в коде (`top_k=10`)

## Отладка

Если нет результатов или мало результатов:
1. Проверьте, что запрос содержит реальные слова из базы
2. Попробуйте более простой запрос (одно-два слова)
3. Проверьте, что Qdrant запущен и содержит данные:
   ```bash
   python -m retriever.search "тестовый запрос"
   ```

Для детальной отладки:
- Смотрите логи в консоли (загрузка моделей, скорость)
- Вывод показывает мин/макс/средний score для сравнения результатов
